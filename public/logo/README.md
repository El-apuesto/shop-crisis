# Logo Assets Folder

This folder contains logo assets for the Crisis Design website.

## Logo Files:
- `logo.png` - Main logo for header
- `logo-icon.png` - Smaller version for parallax component
- `logo-white.png` - White version for dark backgrounds

## Specifications:
- Recommended size: 200x60px for main logo
- Recommended size: 80x80px for icon version
- Format: PNG with transparency
- Background: Transparent
